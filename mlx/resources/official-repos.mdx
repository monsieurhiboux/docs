## Repos Spécialisés

### mlx-swift-examples

[github.com/ml-explore/mlx-swift-examples](https://github.com/ml-explore/mlx-swift-examples) — **2 400 ⭐**

Des apps Swift complètes démontrant l'utilisation de MLX :
- App de chat LLM (iOS + macOS)
- Speech recognition avec Whisper
- Génération d'images
- Embeddings de texte

```swift
// Exemple d'utilisation depuis les examples Swift
import MLXLMCommon
import MLXLLM

let modelConfig = ModelConfiguration(id: "mlx-community/Llama-3.2-3B-Instruct-4bit")
let container = try await LLMModelFactory.shared.loadContainer(configuration: modelConfig)
```

## Documentation officielle

### Référence API MLX

[ml-explore.github.io/mlx](https://ml-explore.github.io/mlx/build/html/index.html)

La documentation officielle couvre :
- **API Python** : `mlx.core`, `mlx.nn`, `mlx.optimizers`
- **Build & Install** : instructions de compilation depuis les sources
- **Metal Kernel API** : pour écrire des kernels GPU custom
- **Tutorials** : exemples pas-à-pas

### Sections clés de la doc

```
mlx/
├── mlx.core          ← Arrays, opérations de base, devices
├── mlx.nn            ← Layers, activations, losses
├── mlx.optimizers    ← SGD, Adam, AdamW...
├── mlx.data          ← Chargement de données
└── mlx.utils         ← Sauvegarde, chargement de poids
```

## Contribuer à MLX

MLX accepte les contributions. Pour commencer :

```bash
# Fork et cloner
git clone https://github.com/votre-username/mlx
cd mlx

# Créer un environnement de dev
cmake -B build -DMLX_BUILD_TESTS=ON -DMLX_BUILD_BENCHMARKS=ON
cmake --build build

# Lancer les tests
cd build && ctest --output-on-failure
```

Les contributions bienvenues incluent : corrections de bugs, nouveaux layers, exemples, documentation, et optimisations de kernels Metal.
