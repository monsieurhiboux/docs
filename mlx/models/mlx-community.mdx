# MLX Community sur Hugging Face

La [MLX Community](https://huggingface.co/mlx-community) sur Hugging Face est l'organisation centrale qui héberge des milliers de modèles pré-convertis et quantisés en format MLX, prêts à tourner sur Apple Silicon.

---

## Qu'est-ce que la MLX Community ?

La MLX Community est une organisation Hugging Face maintenue par la communauté qui :
- **Convertit** les modèles populaires depuis leur format original vers MLX
- **Quantise** ces modèles en 4-bit et 8-bit pour Apple Silicon
- **Valide** le bon fonctionnement de chaque modèle
- **Documente** les spécificités de chaque modèle

Le modèle par défaut pour `mlx_lm.generate` est `mlx-community/Llama-3.2-3B-Instruct-4bit`.

## Comment télécharger et utiliser un modèle

### Via mlx-lm (téléchargement automatique)

```bash
# Le modèle est téléchargé automatiquement
mlx_lm.generate \
  --model mlx-community/Mistral-7B-Instruct-v0.3-4bit \
  --prompt "Bonjour !"
```

Les modèles sont cachés dans `~/.cache/huggingface/hub/`.

### Via huggingface-cli (téléchargement explicite)

```bash
huggingface-cli download mlx-community/Mistral-7B-Instruct-v0.3-4bit \
  --local-dir ./models/mistral-7b-4bit
```

### Via Python

```python
from huggingface_hub import snapshot_download

snapshot_download(
    "mlx-community/Llama-3.2-3B-Instruct-4bit",
    local_dir="./models/llama-3.2-3b"
)
```

## Uploader votre propre modèle fine-tuné

Après un fine-tuning, partagez votre modèle avec la communauté :

```bash
# Fusionner les adapters
mlx_lm.fuse \
  --model mlx-community/Mistral-7B-Instruct-v0.3-4bit \
  --adapter-path ./adapters \
  --save-path ./my-fine-tuned-model \
  --upload-name votre-username/mistral-7b-mon-domaine
```
