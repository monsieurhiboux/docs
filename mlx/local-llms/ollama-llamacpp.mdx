# Ollama & llama.cpp sur Mac

## Ollama — La simplicité avant tout

**Ollama** est l'outil le plus accessible pour faire tourner des LLMs sur Mac. Il gère le téléchargement, la gestion des modèles, et expose automatiquement une API compatible OpenAI.

### Installation

```bash
# Via Homebrew
brew install ollama

# Ou télécharger directement sur ollama.com
```

### Démarrer et utiliser

```bash
# Démarrer le serveur Ollama
ollama serve

# Télécharger et lancer un modèle
ollama run llama3.2
ollama run mistral
ollama run phi3

# Mode interactif
ollama run llama3.2
>>> Bonjour ! Comment fonctionne le fine-tuning ?
```

### Modèles populaires via Ollama

```bash
ollama pull llama3.2          # Llama 3.2 3B (compact, rapide)
ollama pull llama3.1:8b       # Llama 3.1 8B
ollama pull mistral           # Mistral 7B
ollama pull deepseek-r1:8b    # DeepSeek-R1 avec raisonnement
ollama pull qwen2.5:14b       # Qwen 2.5 14B
ollama pull nomic-embed-text  # Modèle d'embeddings
```

### API REST compatible OpenAI

Ollama expose automatiquement une API OpenAI-compatible sur `http://localhost:11434` :

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:11434/v1",
    api_key="ollama"  # clé fictive
)

response = client.chat.completions.create(
    model="llama3.2",
    messages=[
        {"role": "user", "content": "Explique le fine-tuning en 2 phrases"}
    ]
)
print(response.choices[0].message.content)
```

### Intégration dans une app Swift

```swift
import Foundation

struct OllamaClient {
    let baseURL = URL(string: "http://localhost:11434/api")!

    func generate(model: String, prompt: String) async throws -> String {
        let url = baseURL.appendingPathComponent("generate")
        var request = URLRequest(url: url)
        request.httpMethod = "POST"
        request.setValue("application/json", forHTTPHeaderField: "Content-Type")

        let body = ["model": model, "prompt": prompt, "stream": false] as [String: Any]
        request.httpBody = try JSONSerialization.data(withJSONObject: body)

        let (data, _) = try await URLSession.shared.data(for: request)
        let json = try JSONDecoder().decode([String: String].self, from: data)
        return json["response"] ?? ""
    }
}
```

## llama.cpp — Léger et universel

**llama.cpp** est une implémentation C++ ultra-portable de l'inférence LLM. Il fonctionne sur Mac, Linux, Windows, et même sur des Raspberry Pi.

### Installation sur Mac

```bash
# Via Homebrew
brew install llama.cpp

# Ou compiler depuis les sources
git clone https://github.com/ggml-org/llama.cpp
cd llama.cpp
cmake -B build -DLLAMA_METAL=ON  # Activer Metal GPU
cmake --build build --config Release
```

### Télécharger des modèles GGUF

Les modèles llama.cpp sont au format **GGUF** (successeur de GGML). Les meilleures sources :
- [Hugging Face - TheBloke](https://huggingface.co/TheBloke) (millions de téléchargements)
- [Hugging Face - bartowski](https://huggingface.co/bartowski)

```bash
# Via huggingface-cli
pip install huggingface_hub
huggingface-cli download bartowski/Llama-3.2-3B-Instruct-GGUF \
  --include "Llama-3.2-3B-Instruct-Q4_K_M.gguf" \
  --local-dir ./models
```

### Inférence de base

```bash
# Génération simple
./build/bin/llama-cli \
  -m ./models/Llama-3.2-3B-Instruct-Q4_K_M.gguf \
  -p "Bonjour !" \
  -n 200 \
  --gpu-layers 99  # Utiliser tous les layers sur GPU Metal

# Serveur compatible OpenAI
./build/bin/llama-server \
  -m ./models/Llama-3.2-3B-Instruct-Q4_K_M.gguf \
  --port 8080 \
  --gpu-layers 99
```

### Quantifications GGUF disponibles

| Suffixe | Bits | Qualité | Taille (7B) |
|---------|------|---------|-------------|
| Q8_0 | 8-bit | Excellent | ~7 GB |
| Q6_K | 6-bit | Très bon | ~6 GB |
| Q5_K_M | 5-bit | Bon | ~5 GB |
| Q4_K_M | 4-bit | ✓ Recommandé | ~4 GB |
| Q3_K_M | 3-bit | Acceptable | ~3 GB |
| Q2_K | 2-bit | Dégradé | ~2.5 GB |

```swift

dependencies: [
    .package(
        url: "https://github.com/ggml-org/llama.cpp",
        branch: "master"
    )
]
```

import llama

        model = llama_load_model_from_file(modelPath, params)
    }

}
